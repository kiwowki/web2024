01. 요소 메서드 : 선택자 : document.querySelector() : 요소 선택자

document.querySelector()는 웹 페이지에서 HTML 요소를 선택하기 위한 JavaScript 메서드 중 하나입니다. 이 메서드를 사용하면 CSS 선택자를 기반으로 원하는 요소를 선택할 수 있습니다.

{
    // ID가 "myElement"인 요소 선택
    const myElement = document.querySelector("#myElement");
    
    // 클래스가 "myClass"인 요소 선택
    const elementsWithClass = document.querySelector(".myClass");
    
    // 태그 이름이 "div"인 첫 번째 요소 선택
    const firstDiv = document.querySelector("div");
}

선택자 (Selector) : document.querySelector() 메서드는 CSS 선택자를 인수로 받습니다. 이 선택자는 웹 페이지에서 특정 HTML 요소를 찾는 데 사용됩니다. 선택자는 태그 이름, 클래스, 아이디, 속성, 조합 등 다양한 방법으로 요소를 정의할 수 있습니다.
선택된 요소: document.querySelector()는 전달한 선택자와 일치하는 첫 번째 요소를 반환합니다. 일치하는 요소가 없으면 null을 반환합니다.
