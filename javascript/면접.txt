useEffect와 useState는 React 라이브러리에서 주요한 훅(hook) 중 두 가지입니다. React의 함수 컴포넌트 내에서 상태(state)와 부작용(effects)을 다루는 데 사용됩니다.

useState:   (상태 이상 때 변수 관리)

useState는 함수 컴포넌트에서 상태를 관리하기 위한 훅입니다. 
상태는 컴포넌트 내에서 변하는 데이터를 나타내며, useState를 사용하여 상태를 초기화하고 업데이트할 수 있습니다.
예를 들어, 컴포넌트 내에서 특정 변수를 상태로 관리하려면 다음과 같이 사용할 수 있습니다:
const [count, setCount(상태 이상 때 변수 바뀌었는지 감지)] = useState(0);

useEffect:  

useEffect는 부작용(예: 데이터 가져오기, DOM 조작, 구독 및 정리 작업)을 처리하는 데 사용되는 훅입니다.
useEffect 내에서 작성한 코드는 컴포넌트가 렌더링된 후에 실행되며, 렌더링이 발생할 때마다 부작용 코드가 실행될 수 있습니다.
