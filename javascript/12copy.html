01. 퀴즈 이펙트 : 정답 확인하기 유형

주어진 코드는 JavaScript를 사용하여 간단한 퀴즈 애플리케이션을 만드는데 사용되는 코드입니다. 퀴즈를 제출하고 정답 여부를 확인하는 기능을 가진 간단한 웹 페이지를 만듭니다. 사용자가 답을 입력하고 확인 버튼을 누르면 정답 여부에 따라 피드백을 제공하고 정답과 설명을 표시합니다.

{
// 문제 정보
const infoDate = "2021년 10월";
const infoType = "정보처리 기능사";
const infoNum = "1"
const infoQuestion = "CPU의 정보처리 속도 단위 중 초당 100만 개의 연산을 수행한다는 의미의 단위는?"
const infoAnswer = "MIPS"
const infoDesc = "LIPS는 초당 1개, KIPS는 초당 1000개, MIPS는 초당 100000의 연산을 수행하는 것을 의미합니다."

// 선택자
const quizWrap = document.querySelector(".quiz_wrap");
const quizDate = quizWrap.querySelector(".quiz_date");
const quiztype = quizWrap.querySelector(".quiz_type");
const quizNum = quizWrap.querySelector(".quiz_question em");
const quizQuestion = quizWrap.querySelector(".quiz_question span");
const quizAnswer = quizWrap.querySelector(".quiz_answer");
const quizDesc = quizWrap.querySelector(".quiz_desc");
const quizConfirm = quizWrap.querySelector(".quiz_confirm");
const quizInput = quizWrap.querySelector(".quiz_input input");
const quizAnimation = quizWrap.querySelector(".quiz_animation")

//문제 출력
quizDate.innerText = infoDate;
quiztype.innerText = infoType;
quizNum.innerText = infoNum;
quizQuestion.innerText = infoQuestion;
quizAnswer.innerText = infoAnswer;
quizDesc.innerText = infoDesc;

//정답 해설 숨기기
quizAnswer.classList.add("none");
quizDesc.classList.add("none");

// 정답 확인하기
quizConfirm.addEventListener("click", () => {
const userAnswer = quizInput.value;

if(infoAnswer == userAnswer){
quizAnimation.classList.add("like");
} else {
quizAnimation.classList.add("dislike");
}
quizAnswer.classList.remove("none");
quizDesc.classList.remove("none");
quizConfirm.classList.add("none");
quizInput.classList.add("none");
});
}

문제 정보 설정 : const infoDate, infoType, infoNum, infoQuestion, infoAnswer, infoDesc 변수들은 퀴즈의 정보를 저장하는 역할을 합니다.
infoDate는 퀴즈의 날짜를 저장하고, infoType는 퀴즈 유형을 저장하며, infoNum는 문제 번호를 저장합니다.
infoQuestion은 사용자에게 표시될 퀴즈 질문을 저장하고, infoAnswer는 정답을 저장합니다.
infoDesc는 정답 설명을 저장합니다.

선택자 설정 : const 키워드를 사용하여 HTML 문서 내에서 필요한 요소를 선택합니다.
예를 들어, quizWrap, quizDate, quiztype 등의 변수는 HTML 요소를 선택하여 JavaScript 코드에서 사용할 수 있도록 합니다.

문제 출력 : quizDate, quiztype, quizNum, quizQuestion, quizAnswer, quizDesc 변수를 사용하여 퀴즈 정보를 해당 HTML 요소에 출력합니다.
이로써 사용자는 퀴즈의 문제, 유형, 번호, 날짜 등을 볼 수 있습니다.

정답과 설명 숨기기 : 초기에 정답과 정답 설명은 사용자에게 표시되지 않아야 합니다.
quizAnswer와 quizDesc 요소에 none 클래스를 추가하여 숨깁니다.

정답 확인하기 : "확인" 버튼(quizConfirm)을 클릭하면 해당 기능이 실행됩니다.
사용자가 입력한 답을 quizInput에서 가져온 뒤, 이를 userAnswer 변수에 저장합니다.
userAnswer와 infoAnswer를 비교하여 정답 여부를 확인합니다.
정답이면 quizAnimation에 "like" 클래스를 추가하여 긍정적인 피드백을 제공하고, 정답과 설명을 표시합니다.
정답이 아니면 "dislike" 클래스를 추가하여 부정적인 피드백을 제공하고, 정답과 설명을 표시합니다.
