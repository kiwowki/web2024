01. 함수 : 선언적 함수

자바스크립트(JavaScript)에서 변수는 데이터를 저장하는 저장소입니다. 프로그래밍에서 함수는 재사용 가능한 코드 블록으로, 특정 작업을 수행하고 그 결과를 반환하는 데 사용됩니다. 

{
    function func(){
        var x = 100;
        var y = 200;
        var z = "javacript";

        console.log(x);
        console.log(y);
        console.log(z);
    }
    func();
}

선언적 함수(Declarative Function)는 프로그래밍에서 원하는 결과를 얻기 위해 '무엇(What)'을 할지를 정의하는 방식의 함수 스타일을 말합니다. 이 방식에서 개발자는 코드에서 명시적으로 단계별로 작업을 기술하지 않고, 목표를 묘사하고 시스템은 그 목표를 달성하기 위한 내부 동작을 추론하거나 처리합니다.
